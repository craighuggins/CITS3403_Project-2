Welcome

The purpose of this website is to deliver an assessable quiz for users to take. The design is that of a General-purpose quiz such that questions of any category can be used. It is entirely up to the administrator as to the type of questions, as well as the level of difficulty.

In order for the quiz system to function correctly, an Administrator must first set up some questions on the page labelled 'Admin: Add Questions'.
Then, once some questions are set, they can be added to a 'Question Set'. The Question Set is a collection of questions that may belong to a particular theme or category, at the Administrator's discretion.

When a user clicks on the 'Take Test' page, they are presented with a choice of which question set to take. Once they have made a selection from the drop-down menu, clicking the buttom loads the question with multi-choice answer selections.

Upon completing the test, the user can then view the results for the test they just took on the 'Results' page. Assessment of the quiz happens automatically.
They can also view a complete history of all tests they have taken on the 'Test History' page, which shows the user some basic statistics of their test such as the number of questions they got correct, the number incorrect, and their score. It also shows the time the test was taken. Global test statistics can also be viewed on the Question Set Stats page, and admins can view specific user's test results on the Search User Results page!

As the quiz has a general-purpose setup, it can be used in many different scenarios. For example, a University or School could use it as a test for students. Another example could be a social media company using it as a fun pub quiz.
